# Seagull Game (Atracón en Navidad)

This game was designed from scratch for a university project, getting inspiration from the classic Flappy Bird. The goal is to move the character (a young seagull) to help her get food while dodging the city’s ubiquitous Christmas lights, and to win the game, you have to get through all 3 levels without losing all your lives.

## Instructions

If you want to play the game, just download the .exe file :) Please note that the game is only available in Spanish.
